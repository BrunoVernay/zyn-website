How to contribute:

  • If you create instruments, share the instruments/master settings through the mailing list.
  • If you are programmer, you can contribute some code.
  • If you are a general user, you can do a few things:
    • You can report bugs (if no one reports a bug, it is less likely to be found)
    • You can request features (If a feature is not discussed, it will not appear in a new release)
  • If you can donate some money, please do so; this way more time can be spend on ZynAddSubFX in the future. Please notice that there is no obligation to pay for using this program (even for commercial purposes) on conditions of General Public License.
